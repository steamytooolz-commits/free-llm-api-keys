## 🚀 如何使用

所有 Key 兼容 **OpenAI API 格式**，设置 Base URL 并粘贴 Key 即可。

**Base URL：** `https://aiapiv2.pekpik.com/v1`

### cURL

```bash
curl https://aiapiv2.pekpik.com/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer 粘贴Key" \
  -d '{
    "model": "gpt-5.5",
    "messages": [{"role": "user", "content": "你好！"}]
  }'
```

### Python

```python
from openai import OpenAI

client = OpenAI(
    base_url="https://aiapiv2.pekpik.com/v1",
    api_key="粘贴Key"
)

response = client.chat.completions.create(
    model="gpt-5.5",
    messages=[{"role": "user", "content": "你好！"}]
)
print(response.choices[0].message.content)
```

### Node.js

```javascript
import OpenAI from 'openai';

const client = new OpenAI({
  baseURL: 'https://aiapiv2.pekpik.com/v1',
  apiKey: '粘贴Key',
});

const response = await client.chat.completions.create({
  model: 'gpt-5.5',
  messages: [{ role: 'user', content: '你好！' }],
});
console.log(response.choices[0].message.content);
```

## ❓ 常见问题

<details>
<summary><b>这些 Key 是什么？</b></summary>

平台自有签发的 API token（`sk-xxx`），兼容 OpenAI API 格式。一个 Key 可以访问 90+ 模型。
</details>

<details>
<summary><b>和直接用 OpenAI 有什么区别？</b></summary>

| | 本项目 | 直接用 OpenAI |
|---|---|---|
| 需要信用卡 | ❌ 不需要 | ✅ 需要 |
| 免费额度 | ✅ 每个 Key $20-$100 | ❌ 无 |
| 多模型 | ✅ 一个 Key 用所有模型 | ❌ 每家单独注册 |
| 中国可用 | ✅ 直连 | ⚠️ 需要代理 |
| 最新模型 | ✅ GPT-5.5, Claude 4.6, Grok 4.20, Gemini 3.1 | ✅ 仅 OpenAI |
</details>

<details>
<summary><b>有什么限制？</b></summary>

- 每个 Key 预算 **$20 – $100**
- 速率限制：每分钟 3-20 次请求 (RPM)
- 有效期 24-48 小时
- Key 是公开共享的，额度可能被他人消耗
</details>

<details>
<summary><b>Key 不能用了？</b></summary>

Key 是公开共享的，额度可能已被用完。服务器定时任务每天多次发布新 Key，并全天清理失效 Key；稍后回来即可看到最新可用 Key。也可以 **Watch → Releases** 接收通知。
</details>

<details>
<summary><b>安全吗？</b></summary>

- 平台自有 token，不是第三方厂商的 Key
- 每个 Key 有独立预算和速率限制
- 不记录对话内容
- 24-48 小时后自动失效
</details>

<details>
<summary><b>支持哪些地区？</b></summary>

全球可用，中国大陆可直连，无需代理。
</details>
