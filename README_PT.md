## 🚀 Como Usar

```python
from openai import OpenAI

client = OpenAI(
    base_url="https://aiapiv2.pekpik.com/v1",
    api_key="COLE_SUA_CHAVE_AQUI"
)

response = client.chat.completions.create(
    model="gpt-5.5",
    messages=[{"role": "user", "content": "Olá!"}]
)
print(response.choices[0].message.content)
```
